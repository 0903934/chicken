About Us

Who Are We?

We are a 5 person team studying various related IT degrees at the Robert Gordon University in Aberdeen.

What Are We Doing?

As part of our course we were challeneged to create a new and engaging software application. Our idea is a predator detection system which alerts and allows the user to see an icoming predator. For example a fox appearing at a chicken coop.

How Did We Do It?

By using various software techniques. The network used for the detection node is YOLO by Darkweb. The application then uses the Darkflow Python Wrapper. The deteced objects are then highlighted in green (for good, chickens etc) or in red (for bad, foxes etc). Attached to the box is a confidence percentage which allows us to measure how accurate the software is.

A combination of Adobe programs was used to create the GUI interface to allow us to showcase our work.
